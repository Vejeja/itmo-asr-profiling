# Профилирование моделей автоматического распознавания речи (ASR)

Корнилова Валерия Александровна

Этот проект выполнен в рамках дисциплины "Эффективные модели глубокого обучения" программы Ai talent Hub.

## Описание

В рамках проекта проводилось профилирование моделей автоматического распознавания речи (ASR) с использованием различных конфигураций оборудования (CPU и GPU). Анализ включал в себя измерение энергопотребления, времени обработки, точности распознавания (WER) и других метрик.

## Структура проекта

- `ASR_Profiling_TEDLIUM_cpu.ipynb` - Jupyter Notebook с профилированием на CPU
- `ASR_Profiling_TEDLIUM_gpu.ipynb` - Jupyter Notebook с профилированием на GPU
- `ASR_Experiments/` - Директория с результатами экспериментов и CSV-файлами
- `ASR_Experiments/plots/` - Директория с графиками и визуализациями результатов
- `report/` - Директория с отчетом и презентацией

## Результаты

Результаты профилирования включают в себя различные графики зависимости:
- Энергопотребление vs Задержка
- Энергопотребление vs Длина аудио
- Задержка vs Длина аудио
- WER vs Задержка
- WER vs Длина аудио
- Использование памяти GPU vs Размер батча
- Детализация задержки
